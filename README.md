# EventTicket NFT Hub

## Project Title

EventTicket NFT Hub

## Project Description

EventTicket NFT Hub is a decentralized ticketing platform built on the Stellar Soroban smart contract framework. The platform issues event tickets as unique blockchain-based digital assets, allowing secure ownership, transfers, event check-ins, and fraud prevention.

A key feature is the **Clawback Mechanism**, which enables administrators to revoke tickets involved in fraud, unauthorized resale, or payment disputes.

## Project Vision

To provide a transparent, secure, and fraud-resistant event ticketing system powered by blockchain technology.

## Key Features

* NFT-style event tickets
* Secure ticket ownership and transfers
* Event check-in validation
* Anti-fraud clawback mechanism
* On-chain ticket tracking and verification

## Usage Instructions

1. Deploy the contract and set an admin.
2. Mint tickets for an event.
3. Transfer tickets between users.
4. Check in tickets at the event.
5. Claw back fraudulent or invalid tickets.
6. Query ticket information on-chain.
